BOOOTTLE.OOO uses a number of open source projects to work properly:

* [three.js] - JavaScript 3D library.
* [cannon.js] - A lightweight 3D physics engine written in JavaScript.
* [Node.js] - Node.jsÂ® is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. 
* [Express] - Fast, unopinionated, minimalist web framework for Node.js

### Installation
BOOOTTLE.OOO requires [Node.js] v4+ to run.

```sh
$ git clone https://github.com/netrocc/booottle.ooo.git
$ cd booottle.ooo
$ npm install
$ node app.js
```
